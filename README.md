# 远程控制项目

这是一个基于Python的远程控制项目，允许客户端控制服务器端的鼠标和键盘操作。

## 功能特点

- 支持鼠标点击事件
- 支持键盘输入事件
- 实时状态显示
- 简单的图形用户界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 首先在服务器端运行：
```bash
python server.py
```

2. 然后在客户端运行：
```bash
python client.py
```

3. 在客户端输入服务器IP地址（默认为localhost），点击"连接到服务器"按钮

4. 连接成功后，客户端的鼠标点击和键盘输入将会被发送到服务器端执行

## 注意事项

- 确保服务器端和客户端都在同一个网络中
- 服务器端需要允许5000端口的访问
- 使用完毕后请及时断开连接
- 请谨慎使用，确保有适当的权限

## 系统要求

- Python 3.6+
- PyQt6
- pynput
- pyautogui
- pillow
